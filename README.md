Этот репозиторий создан для обучения работе с Git и GitHub, а также для систематизации учебных материалов, расчетов, чертежей и отчетов по курсовым проектам. Здесь будет храниться история всех этапов моей работы.
*   Конспекты лекций.
*   Лабораторные работы по CAD-программам (AutoCAD, Revit).
*   Расчеты в MathCAD и Excel (по механике грунтов, железобетонным конструкциям).
*   Эскизы и чертежи объектов.
*   Отчеты по практикам и курсовым проектам.
*   Полезные скрипты для автоматизации расчетов.
